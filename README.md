# Usage
<!-- usage -->
```sh-session
$ npm install -g nest-clean
$ nest-clean COMMAND

$ nest-clean (-v|--version|version)
nest-clean/1.0.0 win32-x64 node-v12.8.0
$ nest-clean --help [COMMAND]
USAGE
  $ nest-clean COMMAND
...
```
